Projects

Raspberry Pi Infotainment Display

The following project was developed with the idea of using a Raspberry Pi as an infotainment display. While using Rhys Morgans 'react-carplay' project as a base, a menu is displayed on screen. It allows users to use Carplay or have a standby mode. Other features such as 'Track' and 'Reverse Camera' mode will be implemented in the future.

Checkers Game Web Project

Collaborated on a group of two on developing this project. The focus was to implement a website where a Checkers game is hosted, where users can log in and play against each other in the same machine or against a bot. The project was developed using HTML, CSS, JavaScript, and PHP. Log in information is stored in a phpMyAdmin database. The project is not currently hosted on a website but was implemented using XAMPP.

Deep Fake Detection Using Benford's Law

This project was built with two other group members with the help of our professor Dr. Belman. The focus of the project is to build a Deep-Fake image detection model using Benford's Law. Some of the tools used in the project include: a Kaggle Dataset, Python, and Jupyter Notebook.

Nutrition Relational Database

The following project was built using MySQL and a Kaggle Dataset. Our goal was to implement a relational database that stores information about different types of food and their nutritional values. This would allow users to build a meal plan based on their nutritional needs. It also served as a way to practice SQL queries and database management.
